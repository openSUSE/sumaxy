=======================================================================
                           Installing SUMAXY
=======================================================================


SUSE, openSUSE
==============

::

  zypper install cram haveopt make xmlstarlet zsh python-docutils
  git clone https://github.com/openSUSE/sumaxy.git
  mkdir sumaxy-build
  cd sumaxy-build
  ../sumaxy/configure
  make check
  sudo make install
